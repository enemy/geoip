GeoIP lookup

GeoIP city & country lookup using MaxMinds free geolocation database and Sinatra.

Usage

Query

GET /domain.tld
  • Returns HTML
  • Try it: http://geoip.api.enemy.fi/google.com
GET /domain.tld.json
  • Returns JSON
  • Try it: http://geoip.api.enemy.fi/google.com.json
